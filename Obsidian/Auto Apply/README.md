This Obsidian Vault is a collection of markdown files meant to help organize the auto apply vault
It serves the purpose of organizing tasks
Identifying bottlenecks
Serving as a structured area for data retrieval for project context

## Cloning the repo
```
git clone https://github.com/thebarlow/AutoApply.git
```
Navigate into AutoApply/
## Starting the server
```
./start.bat
```
On your browser navigate to localhost:8000
This should start the onboarding process.
### Starting the dev server
The Dev server has live updates on changes to the React Dashboard. Use this when changing the UI actively.
```
auto_apply/start.bat dev
```
This launches the dev server on localhost:5173
